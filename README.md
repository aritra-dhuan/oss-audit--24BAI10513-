# Open Source Audit — Git

## 📌 Project Overview

This project is part of the Open Source Software course. It focuses on analyzing Git as an open-source tool, including its origin, licensing model, Linux footprint, and role in the open-source ecosystem.

Along with the theoretical analysis, the project also includes practical implementation using Bash shell scripts to demonstrate basic Linux and automation concepts.

## 🖥️ Scripts Included

### 1. System Identity Report

Displays system details such as kernel version, user, uptime, and OS information.

### 2. FOSS Package Inspector

Checks whether Git is installed on the system and displays basic package details.

### 3. Disk and Permission Auditor

Analyzes important system directories and shows their size, permissions, and ownership.

### 4. Log File Analyzer

Analyzes a log file (tested using `/var/log/syslog`) and counts occurrences of a keyword like "error".

### 5. Open Source Manifesto Generator

Takes user input and generates a personalized open-source philosophy statement in a text file.

## ▶️ How to Run the Scripts

1. Open terminal

2. Navigate to the project directory

3. Give execution permission:
   chmod +x script_name.sh

4. Run the script:
   ./script_name.sh

### For Script 4 (Log File Analyzer):

./script4.sh /var/log/syslog error
